-- Instala reqs en venv

python3 -m venv venv

source venv/bin/activate



requirements.txt
-----------------
fastapi
uvicorn[standard]
requests
pydantic

pip install -r requirements.txt


Lanza el servidor:

uvicorn main:app --reload --port 8090


Haz una petición POST con curl o Postman:

curl -X POST "http://127.0.0.1:8071/extraer" \
     -H "Content-Type: application/json" \
     -d '{"texto": "Catalunya tiene una población de 8 millones de habitantes y 2 millones de poblacion extranjera"}'
